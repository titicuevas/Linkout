🚀 **¡Presentando Linkeout!** 

¿Eres desarrollador junior y te sientes abrumado con la búsqueda de empleo? Te presento **Linkeout**, una plataforma que he desarrollado para ayudar a organizar y mantener la motivación durante todo el proceso de búsqueda laboral.

### 🎯 **¿Qué hace Linkeout?**

📊 **Diario de Candidaturas Inteligente**
- Seguimiento detallado de todas tus candidaturas
- Estados avanzados (Contacto inicial → Entrevista → Prueba técnica → Contratación)
- Sistema de fechas dual para tracking temporal completo
- Filtros visuales avanzados con búsqueda inteligente

📈 **Dashboard de Estadísticas Visuales**
- Gráficos interactivos para visualizar tu progreso
- Análisis por estado, origen, tipo de trabajo y ubicación
- Métricas motivadoras en tiempo real

🎯 **Motivación IA Personalizada**
- Consejos de diferentes roles (madre, hermano, motivador, psicólogo, etc.)
- Respuestas generadas con Google Gemini AI
- Interfaz empática con iconos y colores temáticos

💪 **Retos de Bienestar**
- Generación automática de retos físicos personalizados
- Tres niveles de dificultad según el puesto y empresa
- Sistema de puntos y motivación

📝 **Comunidad "Desahógate"**
- Espacio seguro para compartir experiencias
- Posts anónimos para apoyo entre desarrolladores

### 🛠️ **Stack Tecnológico**
- **Frontend**: React 19 + Vite + Tailwind CSS
- **Backend**: Node.js + Express + Google Gemini AI
- **Base de Datos**: Supabase (PostgreSQL)
- **Despliegue**: Railway
